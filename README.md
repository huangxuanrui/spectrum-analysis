# 燃烧、退出与消失——中国底层青年现象的谱系分析

2020-2025 年中国互联网底层青年现象的谱系分析。全文约 9000 字。

## 内容

将 40+ 个看似分散的网络热点现象整理为七条主线：

1. **教育就业线** — 从做题到上岸：高考誓师、张锡峰（土猪→水豚）、符文迪（600 分很耀眼）、孔乙己文学、小镇做题家、县中塌陷、县城婆罗门、烂尾娃、全职儿女/全职孙子、假装上班、考公狂潮、张雪峰、张小龙
2. **劳动回报线** — 从打工人到牛马：打工人、牛马文学、外卖骑手、算法劳动、灵活就业、三和大神、马驹桥、35 岁门槛
3. **身体退出线** — 从被看见到不出现：杀马特、精神小伙、社会摇、脆皮年轻人、45 度青年、佛系、积极废人、发疯文学、躺平、水豚/卡皮巴拉、老鼠人
4. **城市空间线** — 从北漂到走线：北漂文化、2017 北京清退、三和与马驹桥的年龄差、鹤岗买房、县城回流、走线、丁胖子与甜甜圈
5. **婚恋家庭线** — 从成家到拒绝再生产：力工梭哈、彩礼与低信任、性压抑理论、四不青年与十不青年、不婚不育与断亲
6. **消费降级线** — 从买更好到花更少：穷鬼套餐、平替文学、反向消费、特种兵旅游、上香与彩票
7. **平台命名线** — 从表达自己到被别人命名：慢脚文化、嘉豪狩猎、苹果安卓论、理论博主的命名游戏、李新野

## 元叙事

系统给底层青年一个承诺 → 承诺变得不稳定 → 信号被放大、包装、变现 → 收到信号的人分化出整个反应谱系：继续冲、低头干、往外跑、往家退、往里缩、降低欲望、被内容消费。
